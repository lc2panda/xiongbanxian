# 梅花易数算法参考（精校版）

本文是对原 tkinter 程序内算法的清理与权威校核，供 SKILL 内部参考。

## 一、先天八卦数（邵雍）

```
乾 1  兑 2  离 3  震 4  巽 5  坎 6  艮 7  坤 8
```

取卦用 `mod 8`，余 0 对应先天数 8（坤）。

| mod8 结果 | 卦 |
|---|---|
| 0 | 坤 |
| 1 | 乾 |
| 2 | 兑 |
| 3 | 离 |
| 4 | 震 |
| 5 | 巽 |
| 6 | 坎 |
| 7 | 艮 |

## 三、起卦公式

### A. 年月日时起卦法（最经典）

公式：
```
上卦 = (年支数 + 农历月 + 农历日) mod 8（余 0 取 8）
下卦 = (年支数 + 农历月 + 农历日 + 时支数) mod 8
动爻 = (年支数 + 农历月 + 农历日 + 时支数) mod 6（余 0 取 6）
```

取数：
- **年数**：取年支，子 1、丑 2、…、亥 12
- **月/日**：以农历数（1–12/1–30）
- **时数**：取时支，子时 1、丑时 2、…、亥时 12

### B. 数字 / 字符起卦

| 位数 | 规则 1（邵雍传统） | 规则 2 |
|---|---|---|
| 1 位 | 上=d%8，下=时支%8，动=(d+时支)%6 | — |
| 2 位 | 上=d1%8，下=d2%8，动=(d1+d2)%6 | — |
| 3 位 | 上=d1%8，下=d2%8，动=d3%6 | 上=d1、下=(d2+d3)%8、动=总和%6 |
| 4+ 位 | 前半段求和=上、后半段求和=下、总和=动 | — |

可选"视 0 为 8"与"动爻加时辰"两项。

### C. 小六壬 三步起课

```
月课索引 = (农历月 - 1) mod 6
日课索引 = (月课索引 + 农历日 - 1) mod 6
时课索引 = (日课索引 + 时支数 - 1) mod 6
```

索引 0-5 对应：大安、留连、速喜、赤口、小吉、空亡。**终断以时课为主**。

## 四、衍生卦

- **互卦**：本卦二三四爻作下卦、三四五爻作上卦
- **变卦**：翻转动爻所在的阴阳
- **错卦**：六爻整体阴阳俱反
- **综卦**：六爻整体上下颠倒

## 五、体用判定

| 动爻位置（自下而上） | 体卦 | 用卦 |
|---|---|---|
| 初爻 / 二爻 / 三爻（在下卦） | 上卦 | 下卦 |
| 四爻 / 五爻 / 上爻（在上卦） | 下卦 | 上卦 |

## 六、五行生克判断表

| 关系 | 定义 | 吉凶 |
|---|---|---|
| 比和 | 体用同五行 | 大吉 |
| 用生体 | 用之五行生体 | 大吉 |
| 体克用 | 体之五行克用 | 小吉 |
| 体生用 | 体生用（外泄） | 小凶 |
| 用克体 | 用克体（外逼） | 大凶 |

五行生序：木→火→土→金→水→木
五行克序：木→土→水→火→金→木
